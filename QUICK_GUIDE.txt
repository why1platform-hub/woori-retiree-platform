## â“ FAQ

**Q: Can regular admins change settings?**
A: No, only SuperAdmin users can access platform settings.

**Q: What if I upload a very large image?**
A: It will work, but database storage will increase. Consider resizing images first.

**Q: Does logo change affect different languages?**
A: No, logo is global. Only name changes per language.

**Q: Can I change the settings from API?**
A: Yes! Use PUT /api/admin/settings with proper authentication.

**Q: What image formats are supported?**
A: PNG, JPG, SVG recommended. Most common formats work.
